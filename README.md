example usage
```
#list complete json file, --r for recursive
py jpower.py config.json --r

#only list elements on given level, -level
py jpower.py config.json -ext
py jpower.py config.json -ext -shell

#also combinable
py jpower.py config.json -ext -shell --r

# to change file[ext][shell][colorscheme] to epic simply append the input value
py jpower.py config.json -ext -shell -colorscheme epic


```
